What Is A System?


A system is a set of things working together as parts of a mechanism or an interconnecting network; a complex whole.



Making Systems


What is the best way to a system? It is probably very dependant on the exact system that is being created. That however does not mean there are not general principles that can be applied to making systems that make things easier or accelerates the process, at least in the vast majority of cases.

When making a system, the goal is to apply changes to it in such a way that it will at some point in the future be able to satisfy set goals and/or discover new goals that can be used to justify better systems.
